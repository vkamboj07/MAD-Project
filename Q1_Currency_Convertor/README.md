# Currency Converter (INR / USD / JPY / EUR)

## Features
- Convert between **INR, USD, JPY, EUR** (offline demo rates)
- Swap currencies
- **Settings** screen to toggle **Light/Dark** theme (persists via SharedPreferences)

## How to run
- Open this folder in **Android Studio**
- Let Gradle sync
- Run the `app` configuration on an emulator/device

## Notes
- Rates are stored in `app/src/main/java/com/example/currencyconverter/Rates.java` and can be updated.
