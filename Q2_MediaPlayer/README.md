# Android_Dev

Android projects for Mobile Application Development (MAD).

## Projects

| Folder | Description |
|--------|-------------|
| Q1/ | Currency Converter — INR, USD, JPY, EUR with dark/light theme |
| Q2/ | Media Player — play/pause/stop audio using MediaPlayer API |

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/vkamboj07/Android_Dev.git
   ```
2. Open the desired project folder (e.g. `Q1/`) in Android Studio.
3. Let Gradle sync, then run on a device or emulator (API 21+).
